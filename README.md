# zingfind/game.py

A python application where you can explore a 3x3 room and interact with it.

## Prerequisites

Python

## Introduction

Run the following:

```
git clone https://github.com/zingfind/game
cd game
python3 ./game.py
```

Use WASD to move your character.

Interact with objects next to you.
